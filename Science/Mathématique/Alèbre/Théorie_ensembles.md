# Théorie des ensembles

## Démonstration

### Inclusion ou égalité

Soit x ∈ E, montrons que x ∈ F : ...

Exemple :

> { x ∈ ℝ | ∃ y ⩾ 0, x ⩾ y } ⊂ ℝ+
