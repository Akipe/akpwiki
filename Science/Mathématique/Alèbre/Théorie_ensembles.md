# Théorie des ensembles

## Démonstration

### Inclusion ou égalité

Soit x ∈ E, montrons que x ∈ F : ...

Exemple :

> { x ∈ ℝ | ∃ y ⩾ 0, x ⩾ y } ⊂ ℝ+

Deux manière de montrer égalité :

#### Double inclusion

Soit x dans E, montront que x est dans F. Puis pour y dans F, montront que y est dans E.

#### Par équivalence

Pour tout x : x ∈ E <=> ... <=> ... <=> x ∈ F
